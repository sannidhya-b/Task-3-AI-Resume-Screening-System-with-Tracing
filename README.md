# 🤖 AI Resume Screening System using LangChain

## 📌 Project Overview
This project is an AI-powered Resume Screening System built using LangChain.  
It evaluates candidate resumes against a job description and provides a fit score along with an explanation.

---

## 🎯 Objective
- Build an intelligent resume evaluation system
- Extract relevant skills and experience from resumes
- Match candidate profiles with job requirements
- Generate a score (0–100) with reasoning

---

## ⚙️ Tech Stack
- Python
- LangChain (LCEL)
- HuggingFace Transformers (distilgpt2)
- Jupyter Notebook

---

## 🔄 Pipeline Architecture

Resume → Skill Extraction → Matching → Scoring → Explanation

---

## 🧠 Features

### ✅ Skill Extraction
- Extracts skills, tools, and experience from resumes

### ✅ Matching Logic
- Compares resume content with job description

### ✅ Scoring System
- Assigns a score based on relevance (0–100)

### ✅ Explainability
- Provides reasoning for the assigned score
